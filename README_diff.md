# Telegram бот для прогноза погоды

Бот предоставляет актуальную погоду для выбранных городов через Telegram.

## Возможности

- 🌤️ Получение текущей погоды для 8 крупных городов России
- 📍 Выбор города через интерактивные кнопки
- 💬 Команда `/weather <город>` для быстрого запроса
- 🌡️ Температура, ощущаемая температура, влажность, скорость ветра
- 🇷🇺 Описание погоды на русском языке

## Доступные города

- Москва
- Санкт-Петербург
- Казань
- Новосибирск
- Екатеринбург
- Нижний Новгород
- Сочи
- Владивосток

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите бота:
```bash
python weather_bot.py
```

## Использование в Telegram

1. Найдите вашего бота в Telegram (по токену или имени)
2. Нажмите `/start` для запуска
3. Выберите город из списка кнопок
4. Или используйте команду `/weather Москва` для быстрого запроса

## Команды

- `/start` - Запустить бота и показать список городов
- `/weather <город>` - Получить погоду для указанного города
- `/help` - Показать справку по командам

## Настройка

Для изменения настроек отредактируйте переменные в начале файла `weather_bot.py`:

- `TELEGRAM_TOKEN` - Токен вашего бота от @BotFather
- `OPENWEATHER_API_KEY` - API ключ от OpenWeatherMap
- `CITIES` - Список доступных городов с координатами

## Требования

- Python 3.8+
- python-telegram-bot 22.7
- requests
